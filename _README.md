# What Have I Done?!

> Keeps track of what you did all day and allows you to create an invoice based on your activity.

## Usage

> Create a task and start the timer. Pause for breaks, and stop when you're done.

## Requirements

- Node v6.9.4
- MongoDB v3.4.1
